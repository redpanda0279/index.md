딥러닝 기반 백혈구 감별계수 자동화

by 시우 | Clinical Pathology x AI

📌 프로젝트 개요

본 프로젝트는 혈액 도말 슬라이드 이미지에서 백혈구를 감별하고 계수하는 과정을 딥러닝 모델(CNN)을 이용해 자동화하는 실험입니다. 백혈구의 주요 4종류(Neutrophil, Lymphocyte, Monocyte, Eosinophil)를 분류하여 진단 효율을 높이는 것이 목적입니다.

📊 데이터셋 & 성능

사용한 데이터는 Kaggle의 "Blood Cell Images Dataset"이며, TensorFlow를 활용한 모델 학습 결과 92% 이상의 정확도를 달성했습니다. 모델은 훈련/검증/테스트 셋으로 분할하여 안정적으로 성능을 평가했습니다.

🧠 사용 기술

  • Python (TensorFlow, Matplotlib, NumPy)
  • Convolutional Neural Network (CNN)
  • Image Classification & Preprocessing
  • Data Augmentation

🙋🏻 시우의 한마디

AI는 비전공자에게도 점점 가까워지고 있습니다. 이 프로젝트를 통해 실제 임상 데이터에서 AI가 어떻게 유용하게 활용될 수 있는지 직접 경험하고자 했습니다. 앞으로도 의료+딥러닝 융합 분야에 도전해보고 싶습니다!
