Diff Count AI 자동화 프로젝트

딥러닝 기반 백혈구 감별계수 기술 리서치

✨ 프로젝트 개요

본 프로젝트는 임상병리학의 핵심 작업 중 하나인 백혈구 감별계수(Diff Count)를 딥러닝 기반으로 자동화하기 위한 기술을 탐구하고 정리한 리서치입니다.

⚖️ 사용 기술

  • CNN 기반 이미지 분류
  • YOLO / Faster R-CNN 객체 탐지
  • U-Net 기반 세포 분할

🔍 주요 데이터셋

  • Kaggle: Blood Cell Images
  • Mendeley Blood Cell Dataset
  • Chula-RBC-12 Dataset

📄 참고 논문

  • Deep Learning for Classification of White Blood Cells (2018)
  • Automatic WBC Detection using YOLOv4 (2022)
  • U-Net: Biomedical Image Segmentation (2015)

🚀 상용화 기술 사례

  • CellaVision – 글로벌 WBC 자동화 분석 시스템
  • 노을 miLab BCM – 한국의 소형 현장 진단 AI 플랫폼
  • Sysmex – 디지털 혈액 분석기술 리더
