Diff Count AI

딥러닝 기반 백혈구 감별계수 자동화 프로젝트

✨ 프로젝트 개요

이 프로젝트는 혈액 도말 슬라이드 이미지로부터 백혈구를 분류하고 계수하는 작업을 딥러닝(CNN, YOLO, U-Net 등) 기반으로 자동화하는 방법을 탐색합니다.

🌐 사용된 데이터셋

  • Blood Cell Images - Kaggle
  • Mendeley WBC Dataset

📊 모델 성능 시각화

🚀 사용 기술 스택

  • TensorFlow (CNN 모델 구현)
  • image_dataset_from_directory 기반 데이터 로딩
  • Matplotlib, Seaborn으로 결과 시각화
