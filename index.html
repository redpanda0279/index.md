Diff Count AI 자동화 리서치

딥러닝 기반 백혈구 감별계수 자동화 기술 정리

✨ 프로젝트 개요

이 프로젝트는 백혈구 감별계수(Diff Count)를 딥러닝으로 자동화할 수 있는 가능성을 탐구합니다. 반복적이고 시간 소모적인 수작업을 AI 기반 분석으로 대체하여, 검사 효율성과 정확도를 높이는 데 목적이 있습니다.

⚖️ 핵심 기술 요약

  • CNN 기반 이미지 분류
  • YOLO / Faster R-CNN 객체 탐지
  • U-Net 기반 이미지 분할

🔍 데이터셋

  • Blood Cell Images (Kaggle)
  • Mendeley Blood Cell Dataset
  • Chula-RBC-12 Dataset

📄 관련 논문

  • "Deep Learning for Classification of White Blood Cells" (2018)
  • "Automatic WBC Detection using YOLOv4" (2022)
  • "U-Net: Biomedical Image Segmentation" (2015)

🚀 상용화 예시

  • CellaVision - AI 기반 WBC 자동화 시스템
  • 노을 miLab BCM - 소형 현장 진단기기 (한국)
  • Sysmex - 자동 혈액 분석 장비 (일본)
